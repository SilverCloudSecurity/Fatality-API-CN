# Render Mode

最后修改时间

25 December 2024

### 简介

`render_mode` 枚举用于切换渲染模式

***

### 枚举值

#### normal

* **类型**: `Field`
* **描述**: 正常不透明渲染。所有多边形将被填充颜色，并且纹理采样将得到完全支持。这是在 `layer` 中渲染形状的默认模式

#### wireframe

* **类型**: `Field`
* **描述**: 线框渲染。这意味着多边形将不再被填充颜色，纹理采样也将无法工作
