# Shadow Dir

25 December 2024

### 简介

`shadow_dir` 枚举用于确定 `add_shadow_line` 方法的阴影方向

***

### 枚举值

#### up

* **类型**: `Field`
* **描述**: 阴影最暗的一侧在底部

#### down

* **类型**: `Field`
* **描述**: 阴影最暗的一侧在顶部

#### left

* **类型**: `Field`
* **描述**: 阴影最暗的一侧在左侧

#### right

* **类型**: `Field`
* **描述**: 阴影最暗的一侧在右侧
