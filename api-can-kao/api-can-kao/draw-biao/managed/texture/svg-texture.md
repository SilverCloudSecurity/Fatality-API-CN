# SVG Texture

最后修改时间

25 December 2024

### 简介

`svg_texture` 类型表示一个 SVG 纹理对象

此类型继承自 `texture` 类型 因此其所有基方法和字段在此类型中也可用

向构造函数传递无效的 SVG 数据将导致崩溃 我们将在未来的更新中修复此问题

***

### 构造函数

#### \_\_call

* **描述**: 构造 SVG 纹理
* **参数**:
  * `sv`: `string` - SVG 文本
  * `h`: `float` - 目标高度 默认为 `0` 且 `0` 表示没有自动缩放
* **返回值**:
  * `svg_texture` - SVG 纹理对象
* **示例**:
