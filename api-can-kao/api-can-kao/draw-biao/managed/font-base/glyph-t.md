# Glyph T

最后修改时间

25 December 2024

### 简介

`glyph_t` 类型表示一个字形对象

***

### 字段

#### codepoint

* **只读**
* **类型**: `int`
* **描述**: 字符 codepoint

#### size

* **只读**
* **类型**: `vec2`
* **描述**: 字形尺寸

#### offset

* **只读**
* **类型**: `vec2`
* **描述**: 字形偏移量

#### uv

* **只读**
* **类型**: `rect`
* **描述**: 纹理图集上的 UV 矩形
