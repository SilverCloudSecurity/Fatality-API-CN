# Control Type

最后修改时间

25 December 2024

### 简介

`control_type` 枚举确定当前控件的类型

***

### 字段

#### default

* **类型**: `Field`
* **描述**: 默认控件 很少会遇到此类型

#### button

* **类型**: `Field`
* **描述**: 按钮控件

#### checkbox

* **类型**: `Field`
* **描述**: 复选框控件

#### child\_tab

* **类型**: `Field`
* **描述**: 子标签控件 仅能在 `tab_layout` 控件中找到

#### color\_picker

* **类型**: `Field`
* **描述**: 颜色选择器控件

#### combo\_box

* **类型**: `Field`
* **描述**: 组合框控件

#### control\_container

* **类型**: `Field`
* **描述**: 带容器的默认控件 很少会遇到此类型

#### group

* **类型**: `Field`
* **描述**: 分组框控件

#### hotkey

* **类型**: `Field`
* **描述**: 热键输入控件

#### hsv\_slider

* **类型**: `Field`
* **描述**: HSB 滑块（饱和度/亮度框 色相滑块和不透明度滑块）

#### label

* **类型**: `Field`
* **描述**: 标签控件

#### layout

* **类型**: `Field`
* **描述**: 布局控件

#### list

* **类型**: `Field`
* **描述**: 列表框控件

#### loading

* **类型**: `Field`
* **描述**: 加载旋转器

#### notification\_control

* **类型**: `Field`
* **描述**: 通知项控件 核心 UI 框架中的类型之一 但在 Fatality 中从未实际使用

#### popup

* **类型**: `Field`
* **描述**: 基本弹出窗口

#### selectable

* **类型**: `Field`
* **描述**: 基本可选项目

#### slider

* **类型**: `Field`
* **描述**: 滑块项 此类型不指示使用的内部值类型

#### spacer

* **类型**: `Field`
* **描述**: 间距控件

#### tab

* **类型**: `Field`
* **描述**: 水平或垂直标签

#### tabs\_layout

* **类型**: `Field`
* **描述**: 专门用于操作标签的布局变体

#### weapon\_tabs\_layout

* **类型**: `Field`
* **描述**: 专门用于操作武器标签的布局变体

#### text\_input

* **类型**: `Field`
* **描述**: 文本输入控件

#### toggle\_button

* **类型**: `Field`
* **描述**: 类似按钮但行为类似于复选框的按钮变体

#### window

* **类型**: `Field`
* **描述**: 基本窗口控件

#### widget

* **类型**: `Field`
* **描述**: 基本控件

#### settings

* **类型**: `Field`
* **描述**: 设置控件
