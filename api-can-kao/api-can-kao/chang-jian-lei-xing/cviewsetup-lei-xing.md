# cview\_setup 类型

_<mark style="color:red;">`cview_setup`</mark> <mark style="color:red;"></mark><mark style="color:red;">类型描述了视图设置参数</mark>_

**字段**

* **fov**
  * **类型**: float
  * **描述**: 当前的 FOV（视场角）
* **fov\_viewmodel**
  * **类型**: float
  * **描述**: 当前视图模型的 FOV
* **origin**
  * **类型**: vector
  * **描述**: 当前视图原点
* **view**
  * **类型**: vector
  * **描述**: 当前视图角度
* **aspect\_ratio**
  * **类型**: float
  * **描述**: 当前视图的宽高比（宽度 / 高度）
